max()

Returns the maximum value of the passed two value arguments. That means, it compares the two passed values with each other and returns the one which is larger.

Function Prototype

max(??number1??,??number2??)

Arguments

Argument Name Data Type Description
??number1?? Integer Number One of the two numbers to compare with.
[required]
??number2?? Integer Number One of the two numbers to compare with.
[required]

Return Value

Description Data Type
Largest number of the passed two values. Integer Number

Examples

The following example prints the result of some example argument values to the terminal as soon as the script is loaded.

on init message("max(2,3) = " & max(2,3)) message("max(1,11) = " & max(1,11)) message("max(-30,4) = " & max(-30,4)) end on
