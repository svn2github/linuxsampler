set_controller()

Changes the value of the requested MIDI controller. This is like sending a MIDI control change event programmatically.

Function Prototype

set_controller(??controller??, ??value??)

Arguments

Argument Name Data Type Description
??controller?? Integer Number MIDI controller number.
[required]
??value?? Integer Number New value for MIDI controller.
[required]

Return Value

Description Data Type
Event ID of the new MIDI control change event. Event ID Number
