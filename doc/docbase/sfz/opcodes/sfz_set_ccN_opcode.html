set_ccN Opcode

Sets a different initial MIDI controller value to the requested MIDI controller.

Opcode Prototype

set_cc??N??=??value??

Parameters

Parameter Name Description
??N?? Defines the MIDI controller which should be altered.
[required]
??value?? Defines the MIDI controller's new initial value (0-127).
[required]

Allowed Sections

Should be used in <control> section.

Examples

Sets the so called "MIDI Expression Controller" (MIDI CC #11) to its maximum value (127).

<control> set_cc11=127

Availabilty

Since LinuxSampler 2.0.0.svn41
